# Expense Record Keeper

1. Description:
This project is a Java console application that helps users manage personal expenses.

2 Features:
- Add expenses
- View expenses
- Update expenses
- Delete expenses
- Save/load data from file
- Export/import CSV

3. OOP Concepts;
- Encapsulation (private fields)
- Inheritance (BusinessExpense extends Expense)
- Polymorphism (method override)

4 How it works;
User interacts with CLI menu to manage expenses.
